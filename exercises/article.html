Web标准由万维网联盟（W3C）制定。

为什么要Web标准？

对于浏览器开发商和 Web 程序开发人员在开发新应用程序时遵守指定的标准更有利于 Web 更好地发展。
开发人员按照 Web 标准制作网页，这样对于开发者来说就更加简单了，因为他们可以很容易了解彼此的编码。
使用Web标准，将确保所有浏览器正确显示您的网站而无需费时重写。
遵守标准的Web页面可以使得搜索引擎更容易访问并收入网页，也更易于访问程序代码（如JavaScript和DOM）。

无障碍

无障碍环境是一个HTML标准的重要组成部分。
Web标准，使其更易于为残疾人士使用Web。

W3C - 万维网联盟

W3C 创建和维护Web标准。
蒂姆·伯纳斯·李（Tim Berners-Lee）是万维网联盟创始人发明者，被称为互联网之父：
The dream behind the Web is of a common information space in which we communicate by sharing information.

W3C创建于1994年10月
W3C是成员组织机构，包括IBM、Microsoft、Apple、Adobe、百度、360和各大浏览器厂商等

最重要W3C标准有：

结构标准：HTML
表现标准：CSS
行为标准：JavaScript
